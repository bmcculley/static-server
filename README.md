# static-server

A HTTP server written in Java to serve static files

Compile:
  javac *.java

Create a jar:
  jar -cfm webserver.jar Manifest.txt *.class

## Javadocs

Some documentation of the public methods can be found [here](https://bmcculley.github.io/staticserver/docs/).
